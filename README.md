- **Minecraft Version:** 1.21 - 1.21.11
- **Server Software:** Paper, Folia or compatible forks
- **Java Version:** 21+

## Building

```bash
git clone https://github.com/NighterDevelopment/smartspawner.git
cd SmartSpawner
./gradlew build
```

The compiled JAR will be available in `build/libs/`
